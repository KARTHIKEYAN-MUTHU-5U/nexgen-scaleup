# NexGen ScaleUp

A premium, brutalist-design landing page for NexGen ScaleUp — a technology parent company that builds interactive, high-performance B2B and B2C applications.

## ✨ Features

- **Brutalist Dark Design** with high-octane yellow (#e4ff00) accents
- **Custom Cursor** with spring-based physics tracking
- **Interactive Flashlight Cards** — spotlight follows your mouse
- **Scroll-driven Animations** — parallax hero, reveal text, progress bar
- **Infinite Marquee** with bidirectional scroll
- **Working Contact Form** with mailto fallback
- **Fully Responsive** — desktop, tablet, and mobile
- **SEO Optimized** — proper meta tags, Open Graph, Twitter cards

## 🛠️ Tech Stack

- **React 19** + **TypeScript**
- **Vite** for blazing-fast builds
- **Tailwind CSS v4** for styling
- **Motion** (Framer Motion) for animations
- **Lucide React** for icons

## 📦 Run Locally

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📂 Project Structure

```
nexgen-scaleup/
├── src/
│   ├── App.tsx          # Main application component
│   ├── main.tsx         # Entry point
│   └── index.css        # Global styles & theme
├── index.html           # HTML entry with SEO meta tags
├── vite.config.ts       # Vite configuration
├── tsconfig.json        # TypeScript configuration
└── .github/workflows/   # GitHub Pages deployment
```

## 🚢 Deployment

This project auto-deploys to GitHub Pages on every push to `main` via GitHub Actions.
